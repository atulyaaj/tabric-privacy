# Tabric - Privacy & Support

This repository hosts the **privacy policy** and **support** information for the [Tabric](https://chromewebstore.google.com/detail/kdfeeoijbcoogiponlffiaocdadgkpio?utm_source=item-share-cb) Chrome extension.

## Privacy policy

Read the full privacy policy in **[PRIVACY.md](PRIVACY.md)**.

Tabric stores sessions, notes, and drafts only on your device. We don’t upload your tabs or notes to any server. If you use the in-app feedback form, only the information you enter is sent to the developer via EmailJS.
